Sıkça Sorulan Sorular

Confusion Matrix, bir sınıflandırma modelinin performansını değerlendirmek için kullanılan bir tablodur.

Confusion Matrix, modelin doğru ve yanlış sınıflandırmalarını göstererek modelin doğruluğunu ve hatalarını analiz etmeye yardımcı olur.

Confusion Matrix, gerçek pozitif, gerçek negatif, yanlış pozitif ve yanlış negatif değerlerini kullanarak modelin performansını değerlendirir.

Kullanıcı, True Positive (TP), True Negative (TN), False Positive (FP) ve False Negative (FN) değerlerini girmelidir.

Sensitivity (TPR), modelin doğru pozitifleri tespit etme oranıdır. TPR = TP / (TP + FN)

Specificity (SPC), modelin doğru negatifleri tespit etme oranıdır. SPC = TN / (FP + TN)

Precision (PPV), modelin pozitif tahminlerinin doğruluk oranıdır. PPV = TP / (TP + FP)

Negative Predictive Value (NPV), modelin negatif tahminlerinin doğruluk oranıdır. NPV = TN / (TN + FN)

Accuracy (ACC), modelin genel doğruluk oranıdır. ACC = (TP + TN) / (P + N)

Matthews Correlation Coefficient (MCC), modelin genel performansını ölçen bir metriktir. MCC = (TP*TN - FP*FN) / sqrt((TP+FP)*(TP+FN)*(TN+FP)*(TN+FN))
